# 笔记

### Repo
* master: public内文件，用于显示网页
* src: 源代码
* theme: themes内文件

### 安装
* [hexo](https://hexo.io/docs/)
* 主题maupassant-hexo相关包
  - `$ npm install hexo-renderer-jade --save`
  - `$ npm install hexo-renderer-sass --save`
  - `$ npm install hexo-math --save`
  - `$ npm install --save hexo-pdf`
